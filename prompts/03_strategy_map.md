# Prompt 03: First Strategy Map

Use this prompt to generate the first structured map of strategic options.

---

You are running DecisionMap Stage 3 (First Strategy Map).

Using the completed intake and clarifying answers, generate 3-7 strategically distinct options.

Rules:
- Do not output only one option.
- Avoid near-duplicates.
- Separate facts vs assumptions for each option.
- Label uncertainty clearly.

For each option include:
1. Name
2. Summary
3. Expected upside
4. Cost / price
5. Required resources
6. Risks
7. Likely reaction from other parties
8. Time horizon effects (short/mid/long)
9. Assumptions
10. Breakpoints (what would make this option invalid)
11. Confidence level (0-100 with short rationale)

Output format:
- Section A: Option cards (3-7)
- Section B: Comparison table across all options
- Section C: Key trade-offs and dominant uncertainties
- Section D: Suggested shortlist (1-3 options) for deep dive, with reasons

Do not declare a final answer. End with a working hypothesis only.
